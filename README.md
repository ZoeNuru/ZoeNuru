# ZoeNuru Website

This repository hosts the public ZoeNuru website on GitHub Pages for the user **@ZoeNuru**.  
It presents the ZoeNuru ecosystem and its pillars: ZoeNuru Precise Diagnostics, ZoeNuru Portable Diagnostics, ZoeNuru Nest, ZoeNuru Digital Hub, ZoeNuru Nexus, and future Academy programs.

## Live site

Once GitHub Pages is enabled on the `main` branch, the site is available at:

https://zoenuru.github.io/ZoeNuru

## Tech stack

- Static HTML and CSS (no backend, no build step)  
- Served via GitHub Pages

## Editing the site

1. Open any `.html` file (for example `index.html`) in this repo.  
2. Click the pencil icon in GitHub to edit.  
3. Commit the changes to `main`.  
4. GitHub Pages rebuilds automatically; refresh the live URL after a few minutes.

## Structure

- `index.html` – ZoeNuru umbrella homepage  
- `precise.html` – ZoeNuru Precise Diagnostics  
- `portable.html` – ZoeNuru Portable Diagnostics  
- `nest.html` – ZoeNuru Nest  
- `hub.html` – ZoeNuru Digital Hub  
- `nexus.html` – ZoeNuru Nexus  
- `academy.html` – ZoeNuru Academy (future)  
- Image files – ZoeNuru logos and branding assets
